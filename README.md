# rn-vertical-slider

A vertical Slider for React Native written entirely in javascript.

## Installation

```sh
npm install rn-vertical-slider
```

## Usage

```js
import RnVerticalSlider from "rn-vertical-slider";

// ...

const result = await RnVerticalSlider.multiply(3, 7);
```
